# Sistema_Fiado
App móvil en Flutter para gestionar el crédito informal ("fiado") en tiendas de barrio. Permite registrar clientes, créditos y abonos, controlar el estado de la cartera, clasificar clientes por nivel de riesgo crediticio y generar alertas automáticas de mora mediante n8n. Base de datos PostgreSQL. Proyecto de grado — Ingeniería de Sistemas.
